NAT 동작 — 사설 IP를 공인 IP로 변환

스마트폰(192.168.0.3)이 구글 서버(8.8.8.8)에 요청을 보내고 응답을 받는 과정

스마트폰
192.168.0.3:5000
MAC: CC:DD:..:33
→
공유기 (NAT)
사설: 192.168.0.1
공인: 59.10.20.30
→
구글 서버
8.8.8.8:443
패킷 현황
대기 중
NAT 테이블
사설 IP:포트 공인 IP:포트 목적지 상태
192.168.0.2:6000 59.10.20.30:40001 8.8.8.8:443 기존 항목
192.168.0.3:5000 59.10.20.30:40002 8.8.8.8:443 신규 추가
단계 0 / 5
스마트폰이 구글에 HTTP 요청을 보내려 한다. 출발지는 사설 IP 192.168.0.3:5000이다. 이 주소는 인터넷에서 라우팅되지 않으므로 공유기가 변환해야 한다.
