Race Condition — count += 1

두 스레드가 동시에 count += 1을 실행하면 무슨 일이 생기나

Thread 1
LOAD reg1 = count
ADD reg1 = reg1 + 1
STORE count = reg1
CPU 레지스터
—
reg1
공유 메모리
0
count
Thread 2
LOAD reg2 = count
ADD reg2 = reg2 + 1
STORE count = reg2
CPU 레지스터
—
reg2
초기 상태
count = 0. 두 스레드가 각각 count += 1을 실행한다. 올바른 결과는 count = 2이어야 한다.
⚡ 비결정성 시뮬레이션 — 1000번 실행 시 결과 분포
경쟁 조건에서는 실행할 때마다 결과가 달라질 수 있다
